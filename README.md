# KUDO-AI-React-App

## System Dependencies

- Node.js v12
- Npm v6

## App Set-Up

Go into project directory and run this command

```
$ npm install
```

Add `.env` file & copy environment variables from `.env.example` file

To start the app

```
 $ npm start
```

You are all setup now


## Production build

For production build run

```
$ npm run build
```
